《粤味厨房 / Cooking Bananas》v19 源码交接说明

一、文件说明
1. index.html
   页面结构入口文件。双击即可在浏览器打开游戏。

2. css/style.css
   样式文件，包含封面、游戏界面、卡牌高亮、动画、提示栏等视觉样式。

3. js/game.js
   核心游戏逻辑文件，包含：
   - 初始化游戏
   - 发牌、市场、冰箱、菜谱、订单、道具逻辑
   - 卡牌点击反馈
   - 市场拿牌 / 存冰箱 / 取冰箱 / 做菜飞行动画
   - 返回主菜单
   - 操作提示、缺食材提示、做菜成功弹窗

4. index.single-file.backup.html
   原始单文件备份版，HTML/CSS/JS/内嵌图片都在一个文件中。若拆分版出问题，可回退此文件。

二、如何运行
方式一：直接双击 index.html，用浏览器打开。
方式二：用 VS Code 打开文件夹后，使用 Live Server 插件运行。

三、基础测试链路
每次修改后请至少测试：
1. 首页 → 开始游戏
2. 点手牌，看高亮、上浮、缩放反馈
3. 点市场牌 → 拿市场，看飞行动画
4. 点手牌 → 存冰箱，看飞行动画
5. 点冰箱牌 → 取冰箱，看飞行动画
6. 点菜谱完成做菜，看食材飞向中央、成功弹窗和分数变化
7. 点返回主菜单，确认能回首页

四、后续迭代建议
1. 不要直接覆盖 v19，复制一份后命名为 v20 / v21 再改。
2. 先修规则逻辑，再改美术。
3. 不要把代码压缩成一行，方便排查 bug。
4. 市场逻辑、冰箱逻辑、订单逻辑请优先对照策划案逐条检查。
5. 目前已经移除香蕉货币/香蕉符号，不要重新加入策划案外的货币设定。

五、当前版本定位
v19 是保底可玩版，适合录屏和答辩演示。它不是商业成品版，后续可以继续优化 UI、卡牌资产和规则完整度。
